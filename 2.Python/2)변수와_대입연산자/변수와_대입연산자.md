## 1. 변수란

: 변수라는 건, '값을 저장하는 공간'을 말합니다.

예를 들어서, number라는 변수에 10이라는 값을 할당하면, 이후에 number라는 이름을 사용해서 10이라는 값을 사용할 수 있게 됩니다.

```python
number = 10
print(3) # 3이 출력된다.
print(number) # number에 저장된 10이 출력된다.
```

앞으로 많이 사용하게 될 print()라는 함수는 값을 화면에 보여주는 역할을 합니다.

숫자, 문자, 변수 등을 괄호 안에 넣으면, 값을 화면에 출력합니다.

위의 예시는, number라는 공간에 10을 저장하고, 화면에 출력한 예시입니다.

## 2. 변수는 언제, 왜 쓸까?

#### 1. 변수는 언제 쓰는 걸까?
: 변수는 값을 저장하고 싶을 때 사용하면 됩니다. 숫자, 문자와 같은 데이터를 저장하고 싶다면, 변수를 사용하면 됩니다.

#### 2. 변수는 왜 쓰는 걸까?
: 변수가 없다고 해서 프로그램이 실행되지 않는 건 아니에요. 그렇다면 변수는 왜 쓰는 걸까요?

변수는 어떤 값을 여러 번 사용해야 하는 경우에, 반복해서 사용하기 쉽게 만들어 줘요.

특히, 내가 저장한 값들을 더하고, 빼고, 곱하는 등의 연산 과정을 거칠 때 변수에 담아 두면 쉽게 바꿀 수 있어요.

나중에 배울 조건문, 반복문과 같은 작업을 수행할 때에는 변수 없이는 프로그램을 하기가 쉽지 않을 겁니다.

또, 변수를 사용하면 프로그램을 쉽게 고칠 수 있고, 내가 만든 프로그램을 쉽게 변형시킬 수 있어요.

## 3. 대입연산자란 뭘까

: 대입 연산자(=)라는 건, 변수에 값을 할당하는 데 사용되는 하나의 기호에요.

예를 들어, "x = 5"라는 코드는 변수(공간) x에 5라는 값을 저장하는 것을 의미합니다.

```python
x = 5 #  x라는 공간(변수)에 5라는 값(숫자)을 저장했어요.
print(x) # x라는 변수(공간) 있는 값(숫자)가 어떤 값을을 가지고 있는지 콘솔(화면)에 보여 줘요.
```

위의 코드에서, '='는 대입 연산자라고 불러요. 대입 연산자를 사용하면, 변수라는 공간에 값을 저장할 수 있게 됩니다.

위 코드의 상황에서, 변수 x는 5라는 값을 저장하고 있어요.

## 4. 변수 이름 짓기
: 지금까지 변수의 이름을 a, b, x와 같은 영어 문자로 지었어요.

파이썬에서는 굳이 영어를 사용할 필요는 없고, 한글을 사용해도 괜찮아요.

다만, 변수의 이름으로 지을 수 없는 몇 가지 형태들이 있어요. 예시를 통해 함께 알아 보도록 해요.

```python
# 변수명으로 지을 수 있는 예시
myAge = 15
yourAge = 13
PI = 3.14
first_Name = "Ryu"
last_Name = 'Hyun Jin'
변수 = 5

print(myAge)
print(yourAge)
print(PI)
print(first_Name)
print(last_Name)
print(변수)

'''
[실행 결과]
15
13
3.14
Ryu
Hyun Jin
5
'''
```
: 파이썬에서 변수명은 문자나 밑줄(_)로 시작해야 해요. 그리고 그 뒤에 문자, 숫자, 밑줄을 포함할 수 있어요.


```python
# 변수명으로 지을 수 없는 예시
2nd_place = 5  # 변수명은 숫자로 시작할 수 없습니다.
my-variable = 10  # 변수명에는 하이픈(-)을 사용할 수 없습니다.
class = "math"  # 예약어는 변수명으로 사용할 수 없습니다.
```
: 그러나, 안 되는 것들의 몇가지 예시가 있어요.
1. 변수명은 숫자로 시작하면 안 돼요
2. 변수명에슨 하이폰(-)을 사용하면 안 돼요. 하이폰을 써야 하는 상황이라면 대신 밑줄(_)을 사용해 보도록 해요.
3. 파이썬 예약어(다른 곳에서 사용하기로 약속되어 있는 이름들)는 사용할 수 없어요.

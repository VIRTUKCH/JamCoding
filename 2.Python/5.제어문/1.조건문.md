## 1. 제어문이 뭘까?

제어문이란, 우리가 컴퓨터에게 "이렇게 해라, 저렇게 해라"라고 지시하는 일종의 명령어예요.

평소에 컴퓨터 프로그램은 위에서부터 아래로 차례대로 일을 처리하는데, 제어문을 사용하면 특정한 조건에서만 일을 하게 하거나, 같은 일을 여러 번 반복하게 할 수도 있어요.

이렇게 해서 우리는 프로그램의 흐름을 마음대로 바꿀 수 있습니다.

이처럼, 프로그램의 흐름을 제어하는 실행문을 제어문이라고 합니다.

이번 시간에는 조건문에 대해서 알아 보겠습니다.

## 3. 조건문이 뭘까?
: 조건문이란, "만약 이런 조건이라면, 이 일을 해라"라고 컴퓨터에게 알려주는 명령어예요.

예를 들어, "만약 오늘 날씨가 비가 오면, 우산을 챙겨라"라는 식으로요.

조건문을 통해 우리는 특정 조건에서만 특정한 동작을 하도록 컴퓨터를 제어할 수 있어요.

조건문에는 'if문'이라는 것이 자주 사용돼요.

'if'는 영어로 '만약에'라는 뜻이죠. 파이썬에서는 이렇게 사용해요:

```python
if 비가_온다:
    우산을_챙긴다
```

여기서 '비가_온다'는 조건이 되고, 이 조건이 참(true)일 때만 '우산을_챙긴다'라는 일을 하게 되는 거예요.

조건문을 사용하면, 우리는 날씨, 시간, 점수와 같은 다양한 조건에 따라 다른 행동을 할 수 있게 프로그램을 만들 수 있답니다.

이렇게 조건문을 잘 활용하면, 훨씬 똑똑하고 유용한 프로그램을 만들 수 있어요!

## 4. [실습] 조건문 사용하기

조건문이란, 프로그램이 어떤 조건을 만족할 때만 특정한 동작을 하도록 하는 명령어예요.

마치 "만약 이 조건이 맞다면, 이 일을 해라"라고 컴퓨터에게 지시하는 것과 같아요.

파이썬에서는 if, else, elif를 사용해서 조건문을 만들 수 있어요.

---

### (1) 조건문의 기본 구조
: if문은 가장 기본적인 조건문이에요. 특정 조건이 참(True)일 때만 명령을 실행하도록 해요.
```python
if 조건:
    # 조건이 참일 때 실행할 명령
```

---

### (2) else문
: else문은 if문의 조건이 거짓(False)일 때 실행할 명령을 정해줘요.
```python
if 조건:
    # 조건이 참일 때 실행할 명령
else:
    # 조건이 거짓일 때 실행할 명령
```

---

### (3) elif문
: elif문은 여러 조건 중에서 하나를 선택해서 실행할 때 사용해요. 'else if'의 줄임말이에요.
```python
if 조건1:
    # 조건1이 참일 때 실행할 명령
elif 조건2:
    # 조건1이 거짓이고, 조건2가 참일 때 실행할 명령
else:
    # 모든 조건이 거짓일 때 실행할 명령
```

## ⭐️조건문 사용 예시⭐️

예시 1. 숫자가 10보다 큰지 확인하기
```python
number = 12
if number > 10:
    print("숫자는 10보다 큽니다.")
else:
    print("숫자는 10보다 크지 않습니다.")
```

예시 2. 여러 조건을 사용하기
```python
age = 18
if age < 13:
    print("어린이입니다.")
elif age < 20:
    print("청소년입니다.")
else:
    print("성인입니다.")
```

## ⭐️예제 문제 풀어보기⭐️
### 🚨사용자에게 숫자나 문자를 입력 받는 방법🚨

예제 문제에서, 우리는 사용자에게 숫자나 문자를 입력 받아서, 조건문을 통해 원하는 문장을 출력하게 만들 거에요. 예제 문제를 풀기 이전에, 어떻게 하면 사용자에게 숫자나 문자를 입력 받을 수 있는지 미리 알아 봐요.

컴퓨터 프로그램에게 사용자의 나이나 다른 정보를 입력받으려면, 우리가 컴퓨터에게 "이제 사용자가 무언가를 입력할 거야. 그 입력을 잘 받아서 기억해둬!"라고 말해줘야 해요.

파이썬에서는 이렇게 사용자로부터 정보를 입력받기 위해 input()이라는 함수를 사용해요. 함수란, 특정한 일을 수행하는 명령어 집합이에요.

예를 들어, 컴퓨터에게 "당신의 나이를 입력해주세요"라고 묻고 싶다면, 우리는 input() 함수를 사용하고, 괄호 안에 "당신의 나이를 입력해주세요"라는 안내 문구를 넣어줄 수 있어요.

그럼 컴퓨터 화면에 그 문구가 나타나고, 사용자는 거기에 답을 입력할 수 있어요.

하지만, 여기서 한 가지 중요한 점이 있어요. 컴퓨터는 사용자가 입력한 것을 기본적으로 '글자'(문자열)로 받아들여요.

예를 들어, 사용자가 숫자 10을 입력했다 하더라도, 컴퓨터는 그것을 숫자가 아닌 '1'과 '0'이라는 글자가 나란히 있는 것으로 생각해요.

그래서 우리가 숫자로 계산하거나 비교하려면, 이 '글자'를 진짜 숫자로 바꿔줘야 해요. 이때 사용하는 것이 int() 함수예요. int()는 괄호 안의 글자를 숫자로 바꿔주는 역할을 해요.

```python
a = int(input("당신의 나이를 입력해주세요: "))
```

이렇게 하면, 컴퓨터는 먼저 "당신의 나이를 입력해주세요:"라고 묻고, 사용자가 입력한 나이를 글자에서 숫자로 바꾼 다음, 그 숫자를 a라는 곳에 저장해요.

그래서 나중에 우리가 a를 사용하면, 사용자가 입력한 나이 숫자를 사용할 수 있게 되는 거죠.

이해가 되셨나요? 이렇게 사용자로부터 정보를 입력받고, 그것을 우리가 원하는 형태로 바꿔서 사용하는 것이 컴퓨터 프로그래밍의 기본 중 하나랍니다!

## 5. [생각해 보기] 조건문은 언제 써야 할까?

예제 문제를 푸느라 수고 많았어요. 문제를 풀면서, 조건문이라는 걸 어떻게 사용하는지에 대해 어느 정도 감을 잡지 않았을까 싶어요.

이제 기초적인 개념은 어느 정도 마무리 된 상태이니, 앞으로는 '백준'이라는 온라인 사이트에 들어가서, 조건문과 관련된 여러 문제들을 풀어 볼 거에요.

하지만, 백준에 있는 문제를 풀기 위해서는 알아야 할 것들이 남아 있어요. 바로 '언제 조건문을 써야 하는가'에 대한 생각이에요.

### 조건문은 언제 써야 할까?
: 조건문은 '프로그램의 흐름을 조건에 따라 분기하고 싶을 때' 사용해요. 쉽게 얘기하면, 특수한 상황에 특수한 코드를 작동시키기 위해 사용한다고 볼 수 있어요.

그렇다면, '특수한 상황'인 걸 컴퓨터에게 어떻게 하면 설명할 수 있을까요? 바로 '조건'이라고 하는 걸로 알려 줘야 해요.

'조건'은 '참(True)'과 '거짓(False)'으로 분명하게 나뉘는 질문이나 명제입니다. 반드시 '참'이거나 '거짓'이어야만 해요.

---

예를 들어, 학교에 가는 길에 아이스크림 가게 앞을 지난다고 생각해 봐요.

여기서 "만약 내가 용돈이 1000원 이상 있다면, 아이스크림을 사 먹자"라는 생각을 할 수 있어요

 이때, "내가 용돈이 1000원 이상 가지고 있는가?"를 결정하는 것이 바로 '조건'이 되고, 이 조건에 따라 아이스크림을 사 먹을지 말지를 결정하게 됩니다.

컴퓨터 프로그램에서도 이와 같이 '조건'을 사용해 상황에 따라 다른 작업을 할 수 있습니다.

예를 들어, 게임에서 "만약 플레이어의 체력이 0 이하라면, 게임 오버 처리를 하자"라고 할 수 있어요. 여기서 "플레이어의 체력이 0 이하인가?"가 바로 조건이 되는 겁니다.

이렇게 조건문은 특정 조건이 참인지 거짓인지를 확인하고, 그 결과에 따라 다른 명령을 실행하는데 사용됩니다. 조건이 참(True)이면 하나의 작업을, 거짓(False)이면 다른 작업을 하도록 프로그램을 만들 수 있습니다.

```python
용돈 = 1200
print("용돈이 1000원이 넘는가? : "용돈 >= 1000)

if 용돈 >= 1000:
    print("아이스크림을 산다!")
else:
    print("아이스크림을 사지 않는다.")
```

이 코드에서는 용돈이 1000원 이상인지를 조건으로 확인하고 있어요. 만약 용돈이 1000원 이상이라면 "아이스크림을 산다!"라고 출력하고, 그렇지 않다면 "아이스크림을 사지 않는다."라고 출력해요.

조건문을 사용하는 이유는 경우에 따라 다르게 풀이하기 위해서입니다. 실제 세계에서 우리가 상황에 따라 다르게 행동하는 것처럼, 프로그램도 다양한 상황에 맞춰 다른 반응을 보일 수 있게 해주는 거죠.

이제 '백준' 같은 사이트에서 문제를 풀 때, 이런 조건문을 어떻게 사용하면 좋을지 생각해 보면서 연습해 보면 되겠습니다.

문제의 상황을 잘 읽고, 어떤 조건을 체크해야 할지, 그리고 그 조건에 따라 어떤 작업을 해야 할지를 고민하면서 문제를 풀어 보면 되겠습니다.

## [백준 온라인 저지 사이트](https://www.acmicpc.net/step/4)

매 수업이 끝난 후에는 오늘 배운 내용들을 다시 한 번 정리하는 시간을 가져볼 거예요. 오늘 배운 것들을 직접 설명해 보는 연습도 해보려고 해요.
